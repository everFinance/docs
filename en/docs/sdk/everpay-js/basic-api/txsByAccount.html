txsByAccount

Function#

Get the transactions corresponding to account on everPay by paging

Parameter#

TxsByAccountParams

Return#

Return Type#

TxsResult

Return Fields#

Field information can be viewed in Guide - Dive - Transaction - Transaction Record

Example#

const everpay1 = new Everpay({ debug: true })everpay1.txsByAccount({ account: '0x26361130d5d6E798E9319114643AF8c868412859', page: 1 }).then(console.log)
