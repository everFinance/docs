# 介绍
everpay-js 作为 everPay 客户端开发工具，方便开发者使用 nodeJS、web 端来进行 everPay 集成，来进行 everPay 的支付等。

当前 everPay 账户支持 ethereum, arweave 账户模型，开发者可使用 ethereum, arweave 钱包来访问和操作 everPay 账户。
