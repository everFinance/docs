# 介绍

everpay-js 作为 everPay 客户端开发工具，方便开发者使用 nodeJS、web 端来进行 everPay 集成，来进行 everPay 的支付等。

当前 everPay 账户支持智能账户模型、Ethereum, Arweave 账户模型，开发者可使用

- 邮箱地址
- 或者 Ethereum, Arweave 钱包

来访问和操作 everPay 账户。
