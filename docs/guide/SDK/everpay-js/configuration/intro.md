# 介绍

everpay-js 创建 Everpay 实例的配置项，配置项包含：`debug`, `account`, `chainType`, `ethConnectedSigner`, `arJWK`。
* 配置项类型为 [`Config`](../types#config)。

```ts
export interface Config {
  debug?: boolean
  account?: string
  chainType?: ChainType
  ethConnectedSigner?: Signer
  arJWK?: ArJWK
}
```
