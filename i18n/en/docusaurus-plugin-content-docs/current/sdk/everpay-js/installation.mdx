# Installation and usage

## Installation

<Tabs>
<TabItem value="yarn" label="yarn">

```bash
yarn add everpay
```

</TabItem>
<TabItem value="js" label="npm">

```bash
npm install everpay
```

</TabItem>
</Tabs>

## Usage

### es module
```js
import Everpay from 'everpay'
const everpay = new Everpay()
```

### commonjs
```js
const Everpay = require('everpay')
const everpay = new Everpay()
```
